09:46:59RESEARCH
In order to step down a bit I've decided to retire all work on the game until at least the end of this week, aside from doing my monthy Game Jolt news, and opening the open alpha.

In the meantime I do not want to sit completely still, but you can't expect me to do very much, though.

A game that was planned for quite some time is Mörker, which will be a point-and-click-adventure. I was not 100% sure wether to use the LAURA II engine for that or Love 0.10

This will require me to make a Kthura exporter, however, Kthura relies very much on tiled areas... something Love does not support. I've been looking into alternate ways to create the same effect in Love2D and I think I found it, however it can be quite hard to set this alright, not to mention is that the "bypass" routine I'll need to set up in Love will be quite costly on the RAM memory. Since point-and-click adventures do not really rely on huge maps, this is not necesarily an issue, but I still need to make sure this will not threaten the game. Once the engine has been chosen, there is no way back unless you rewrite everything you have from scratch. My main reason to use Love in stead of LAURA II is Linux support, and Linux support is not worth it for me to settle for less functionality than LAURA II offers ruining the game, and this is one of the main issues I'm dealing with, when it comes to that.
