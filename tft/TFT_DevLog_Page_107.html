Yes, these are all the same icons as used in the original game, so basically I only needed to copy-and-paste them. Lazy,eh?

Nope, I did this for a reason.

You see these icons were designed for the original fairy tale game years ago, and it took me quite some time to make them and they turned out pretty good, if I may say somyself. Ain't nostalgia great sometimes ;)
