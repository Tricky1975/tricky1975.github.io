13:47:37ART
Yes, these are all the same icons as used in the original game, so basically I only needed to copy-and-paste them. Lazy,eh?

Nope, I did this for a reason.

You see these icons were designed for the original fairy tale game years ago, and it took me quite some time to make them and they turned out pretty good, if I may say somyself. They were made in (you'd never believe it) DeluxePaint. The PC (MS-DOS) version that is. I don't know if it worked in Windows back then or if I used DosBOX for that, but that doesn't matter. :)

These icons have become a bit "iconic" for The Fairy Tale. That was the reason I also used them in Dyrt which was originally intended as "The Fairy Tale II". And that is why they are now used in this remake of the game again, in order to bring some "atmosphere" of the original game back.

Ain't nostalgia great sometimes ;)
