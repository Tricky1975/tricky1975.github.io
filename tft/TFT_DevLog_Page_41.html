07:57:28FAILURE
Well, when it comes to Star Story.... I must be very harsh on this now. If your screen is too small for the launcher.... Buy a bigger one. And that goes most of all for Windows users.

I've tried to adapt the launcher to make it compatible for smaller screens, but that only resulted into Windows refusing to draw any of the Window's content.

How do I know this is a bug in Windows itself?

Easy, two reasons make that perfectly clear.
a) On Mac this works PERFECTLY. and b) Dragging the window beyond the edge and dragging it back misshapes it. That only happens when Windows does see a Window as existing, but does not perform any graphic action on it anymore at all.

Well there's more evidence, but these are the two most important reasons.

Now in theory it should be possible to adapt the Launcher The Fairy Tale uses on Star Story as well, but before I get onto that, if first want to know if that launcher does indeed work on smaller monitors.

Since that launcher auto-adapts to the sizes of your screen automatically it should, but when you know MicroSoft as well as I do, you know that what should rarely actually happens.
